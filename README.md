# PolyBot

C'est **un bot discord multi-fonction** privé mais **open-source** :
- Modération
- Musique
- Jeux
- Système de niveaux
- Intégration API

Au fil du temps, j'ai été ammené à implémenter mon bot avec différentes libraries :

| Librairie                                                              | Langage | Maintenu |
|------------------------------------------------------------------------|---------|----------|
| [javacord](https://javacord.org/)                                      | Java    | ✅        |
| [discord.py](https://discordpy.readthedocs.io/en/stable/)              | Python  | ❌        |
| [hikari-lightbulb](https://hikari-lightbulb.readthedocs.io/en/latest/) | Python  | ❌        |
